# Contributing to FlowPOS

Thank you for your interest in contributing to FlowPOS!

## License
By contributing to FlowPOS, you agree that your contributions will be licensed under the same MIT License that covers this project. See the [LICENSE](./LICENSE) file for details. Your name will appear in the git commit history as a recognized contributor.

## How to contribute
1. Fork the repository
2. Create a branch: `git checkout -b feat/your-feature-name`
3. Make your changes
4. Commit clearly: `git commit -m "feat: add barcode scanner support"`
5. Push and open a Pull Request

## Commit message format
feat: new feature | fix: bug fix | docs: documentation | chore: maintenance | test: tests

## Areas where help is most needed
- UI/UX improvements to the POS cashier screen
- Additional payment method integrations
- Translations and localization
- Unit and integration tests
- Documentation

## Questions?
Open an issue and tag it with the `question` label.
